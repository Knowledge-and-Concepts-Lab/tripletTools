# tripletTools for R

This repo contains an R package designed to aid in the analysis of data
from triadic comparisons (tripelt) tasks.

To install, in R 3.5 or greater, do the following:

    install.packages("remotes")
    remotes::install_github("Knowledge-and-Concepts-Lab/tripletTools", build_vignettes = TRUE)

Once installed you can invoke the package as usual with:

[`library(tripletTools)`](https://knowledge-and-concepts-lab.github.io/tripletTools/)

To get a sense of how to use the tools, run
[`vignette()`](https://rdrr.io/r/utils/vignette.html) within an R
session.

All functions are documented in tripletTools_manual.pdf
